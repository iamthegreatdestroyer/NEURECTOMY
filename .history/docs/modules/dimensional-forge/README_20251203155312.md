## Key Components

### 3D Agent Modeler

The core modeling system provides:

- **Component Nodes** - Visual representation of agent components
- **Connection Wires** - Show relationships and data flow
- **Hierarchy Layers** - Organize components by level
- **Dependency Flows** - Visualize dependencies between components

### 4D Timeline Engine

Navigate through time with your agent:

| Mode | Description |
|------|-------------|
| **Replay** | Step through historical agent decisions |
| **Live View** | Real-time agent activity visualization |
| **Simulation** | Run hypothetical scenarios |

## Usage Examples

### Creating a 3D Agent Model

```python
from neurectomy.forge import AgentSculptor

# Create a new agent visualization
sculptor = AgentSculptor()

# Add components
core = sculptor.add_node("core_reasoning", type="processor")
memory = sculptor.add_node("long_term_memory", type="storage")
skills = sculptor.add_node("skill_collection", type="container")

# Connect components
sculptor.connect(core, memory, type="bidirectional")
sculptor.connect(core, skills, type="access")

# Generate 3D model
model = sculptor.render()
model.export("agent_architecture.gltf")
```

### Timeline Navigation

```python
from neurectomy.forge import TimelineEngine

# Create timeline for an agent
timeline = TimelineEngine(agent_id="elite_sentinel")

# Navigate to a specific point
timeline.goto(timestamp="2025-12-01T14:30:00Z")

# Replay decisions
for decision in timeline.replay(duration="1h"):
    print(f"Decision: {decision.action} at {decision.timestamp}")
    
# Run what-if scenario
scenario = timeline.what_if(
    modification={"input": "alternative_data"},
    from_timestamp="2025-12-01T14:30:00Z"
)
```

### Generating from Description

```python
from neurectomy.forge import GenerativeDesigner

# Describe your agent
description = """
A multi-modal reasoning agent with:
- Vision processing capabilities
- Natural language understanding
- Long-term memory with vector search
- Tool calling for external APIs
"""

# Generate 3D architecture
designer = GenerativeDesigner()
model = designer.generate(description)

# View in 3D
model.visualize()
```

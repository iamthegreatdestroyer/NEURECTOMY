## Usage Examples

### Training a Custom Model

```python
from neurectomy.foundry import ModelTrainer, Dataset

# Load dataset
dataset = Dataset.from_huggingface("agent-conversations")

# Configure trainer
trainer = ModelTrainer(
    model_type="transformer",
    task="agent-reasoning",
    dataset=dataset
)

# Train with AutoML
results = trainer.train(
    auto_tune=True,
    tune_metric="agent_accuracy",
    gpus=4
)

# Deploy best model
trainer.deploy(
    target="triton",
    registry="elite-models",
    version="1.0.0"
)
```

### Fine-Tuning a Foundation Model

```python
from neurectomy.foundry import FineTuner

# Configure fine-tuning
tuner = FineTuner(
    base_model="meta-llama/Llama-3-70B",
    method="qlora",
    lora_r=64,
    lora_alpha=16
)

# Fine-tune on agent data
tuner.train(
    dataset="elite-agent-examples",
    epochs=3,
    learning_rate=2e-4
)

# Merge and export
tuner.merge_and_export("elite-sentinel-llm")
```
